## Release History

<Update label="2024-10-15" description="v2.1.0" tags={["feature", "improvement"]}>

## New Features

- Added real-time collaboration with live cursors and inline comments for multiple users.
- Introduced workflow approvals for document reviews before publishing.

## Improvements

- Enhanced search with full-text indexing across spaces and tags.
- Optimized editor performance for large documents, reducing load times by `30%`.

## Bug Fixes

- Fixed import issues with certain Markdown variants from external sources.
- Resolved permission glitches when sharing spaces with SSO users.

</Update>

<Update label="2024-09-20" description="v2.0.0" tags={["feature", "breaking"]}>

## New Features

- Launched space-based organization with nested folders and advanced permissions.
- Integrated API for programmatic document management and automation.

## Breaking Changes

- Updated editor to new Markdown parser; review custom syntax for compatibility.
- Deprecated legacy import formats; migrate to new standardized process.

## Improvements

- Added mobile-responsive design for editing on the go.
- Improved notification system with customizable alerts via email or Slack.

</Update>
